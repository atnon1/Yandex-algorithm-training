# Yandex-algorithm-training
### Решения заданий в рамках [курса по алгоритма](https://yandex.ru/yaintern/algorithm-training) от Яндекса

Курс включает в себя:
* Лекции по алгоритмам
* Домашние задания по темам лекций на Яндекс.Контест
* Разбор домашних заданий

Темы:
1. Сложность, тестирование, особые случаи
1. Линейный поиск
1. Множества
1. Словари и сортировка подсчётом
1. Префиксные суммы и два указателя
1. Бинарный поиск
1. Сортировка событий
1. Деревья
